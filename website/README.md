# Control Center CMS - Professional Website

Eine moderne, professionelle Website für das Control Center CMS mit dunklem schwarz-rot Design.

## Features

- **Next.js 14** mit TypeScript
- **Tailwind CSS** für responsives Design
- **Framer Motion** für Animationen
- **Lucide React** für Icons
- Professionelles dunkles Design mit rot-schwarzem Farbschema
- Vollständig responsive für alle Geräte
- SEO-optimiert
- Performance-optimiert

## Technologie-Stack

- Next.js 14 (React Framework)
- TypeScript
- Tailwind CSS
- Framer Motion (Animationen)
- Lucide React (Icons)

## Design-Highlights

- **Dunkles Professionelles Theme**: Schwarz-rote Farbpalette
- **Glasmorphismus-Effekte**: Moderne UI mit Blur-Effekten
- **Glow-Animationen**: Rote Leuchteffekte für interaktive Elemente
- **Smooth Scrolling**: Flüssige Übergänge zwischen Sektionen
- **Responsive Grid Layout**: Optimiert für Desktop, Tablet und Mobile

## Installation & Development

```bash
# Abhängigkeiten installieren
cd /home/alex/control-center-3_2/website
npm install

# Development Server starten
npm run dev

# Build für Production
npm run build

# Production Server starten
npm start
```

Die Website läuft dann auf `http://localhost:3000`

## Deployment

Die Website kann auf jeder Platform deployed werden, die Next.js unterstützt:
- Vercel (empfohlen)
- Netlify
- AWS
- Custom Server

## Anpassungen

Die Website ist vollständig anpassbar:
- **Farben**: In `tailwind.config.js` anpassen
- **Content**: In `app/page.tsx` bearbeiten
- **Styling**: In `app/globals.css` erweitern
- **Komponenten**: Neue React-Komponenten hinzufügen
