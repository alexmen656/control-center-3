# Template Module

Dieses Template-Modul dient als Vorlage für neue Module.

## Verwendung

1. Kopiere den gesamten `_template` Ordner
2. Benenne ihn um (z.B. `mein-modul`)
3. Passe die Dateien an:
   - `dashboard.provider.ts` - Definiere deine Widgets
   - `index.ts` - Ändere Modul-Name und ID
   - `routes.ts` - Definiere deine Routes
   - `components/TemplateView.vue` - Erstelle deine Hauptansicht

## Dateien

- **dashboard.provider.ts** - Dashboard Widgets
- **index.ts** - Modul Initialisierung
- **routes.ts** - Modul Routes
- **components/TemplateView.vue** - Hauptansicht
- **README.md** - Diese Datei

## Wichtig

Ändere alle Vorkommen von "template" in deinen Modul-Namen!
